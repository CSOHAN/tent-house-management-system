# Tent House Management System (THMS)

## Project Vision

### Purpose

Develop an enterprise-grade Tent House Management System to digitize and automate the day-to-day operations of a tent house business.

The application will help manage:

- Customers
- Bookings
- Inventory
- Payments
- Employees
- Vehicles
- Reports
- AI-assisted operations

---

## Problem Statement

Currently, the business manages operations manually, making it difficult to:

- Track bookings
- Prevent double bookings
- Manage inventory
- Track payments
- Generate reports
- Maintain customer history

---

## Vision

Build a scalable, production-ready backend application using modern Java technologies that can eventually evolve into a SaaS platform for multiple tent house businesses.

---

## Target Users

- Business Owner
- Manager
- Employees
- Accountant
